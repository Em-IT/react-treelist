# react-treelist

A React treelist component to display data in tree structure.

## Installation

```sh
npm install --save react-treelist
```

## Usage

```js
import TreeList from 'react-treelist';

<TreeList
  data={DATA}  
  columns={COLUMNS}
  options={OPTIONS}
  id={'id'}
  parentId={'parentId'}></TreeList>
```
